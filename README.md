# Laravel Blog  
Laravel と Bootstrap を使用して作成した SNS 風ブログアプリです。  
Laravel UI を利用したログイン機能、投稿機能、プロフィール編集などができます。

## 主な機能
- ユーザーログイン（Laravel UI使用）
- 投稿の作成・編集・削除
- コメント機能
- プロフィール表示・編集
- 自分の投稿一覧ページ  

## 使用技術
- Laravel
- PHP
- Bootstrap
